De combinatie van functionele en technische expertise maakt het mogelijk om uw specifieke situatie snel en op de juiste wijze te ondersteunen.

Technologie

In onze projecten maken wij gebruik van state-of-the-art open source technologie zoals EMF, JSF en Hibernate. Software oplossingen worden gemodelleerd met behulp van XML Schema en Object-Orientatie technieken. Springsite levert internet/intranet oplossingen en maakt hierbij gebruik van Apache Tomcat.

Onze oplossingen worden op Linux, Solaris en Windows gebruikt, onze Java oplossingen werken op alle belangrijke databases.

Wij hebben zeer uitgebreide ervaring met XML en XML transformaties (XSLT). XML wordt bijvoorbeeld gebruikt voor leveranciersintegratie. Ook passen wij XML/XSLT toe voor user interface generatie.

eBusiness, inkoop, productie en logistiek

Springsite beschikt over uitgebreide expertise op het gebied van inkoop en logistieke processen. Deze expertise is gebruikt in de ontwikkeling van verschillende standaard eProcurement applicaties en maatwerk eBusiness projecten.
