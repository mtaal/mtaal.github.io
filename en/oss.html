Springsite is an active contributor to several Open Source projects in the Eclipse eco-system.

Texo is the latest model-driven solution which we initiated. Texo is based on our experience with developing ERP-business web applications using javascript front-ends and a metadata driven web service backend. Texo supports Eclipse Modeling Framework modeling concepts in (web)server environments through JPA-annotated code generation and auto-generated full CRUD REST JSON and XML web service functionality in a model-driven runtime solution.

Teneo is one of our earlier model-driven solutions. It is a Model-Relational mapping and runtime database persistence solution for the Eclipse Modeling Framework (EMF). Teneo integrates EMF with Hibernate. We also developed a Teneo variant which works as part of the CDO solution.

For the latest information on Texo and Teneo check out our blog.
